📖 О проекте

Этот сайт создан для изучения возможностей бесплатного хостинга от GitHub. Здесь я буду публиковать свои эксперименты и учебные проекты.

🛠️ Технологии

  • HTML5
  • CSS3
  • GitHub Pages

⚙️ Настройка GitHub Pages

  1. Перейдите в свой репозиторий на GitHub.
  2. Нажмите Settings (вкладка в верхнем меню).
  3. В левом меню выберите Pages.
  4. В разделе Branch выберите ветку (обычно main или master). Оставьте папку /(root).
  5. Нажмите Save.
  6. Через пару минут сайт станет доступен по ссылке, которая появится в этом же разделе.

Важно: Если репозиторий не username.github.io, сайт будет по адресу username.github.io/имя-репозитория.

✅ Проверка и публикация

  • После сохранения настроек подождите 1–2 минуты.
  • Откройте браузер и перейдите по вашему адресу.
  • Если видите свою страницу — всё работает!

💡 Дополнительные советы по красивому оформлению

  • Используйте готовые CSS-фреймворки (Bootstrap или Tailwind CSS).
  • Добавьте иконки с помощью Font Awesome.
  • Сделайте адаптивный дизайн.
  • В README используйте эмодзи, таблицы, списки и выделение кода.

❓ Что делать, если сайт не открывается?

  • Убедитесь, что репозиторий публичный.
  • Проверьте ветку в настройках Pages.
  • Посмотрите вкладку Actions — там могут быть ошибки сборки (для простого HTML сборка не требуется).

🎨 Альтернативный путь: использование темы Jekyll

Если вы хотите ещё более стильный сайт без написания CSS, в настройках Pages выберите тему из галереи (кнопка Choose a theme). GitHub автоматически сгенерирует сайт на основе вашего README.md и добавит стили.
